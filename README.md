# SmartPath Java CRUD System

A Java application implementing CRUD (Create, Read, Update, Delete) functionality 
for managing a tutoring system. The project includes both a GUI interface and 
a console-based interface, with data stored in text files.

## Login Details:
- Username : Staff
- Password: Password1!

## Features

### Student Management
- Add students
- Edit student details
- Remove students
- Search for students

### Tutor Management
- Add tutors
- Edit tutor details
- Delete tutors
- Search for tutors

### Booking Management
- Add bookings
- Edit bookings
- Remove bookings
- Search bookings

### Subject Management
- Add subjects
- Edit subjects
- Delete subjects
- Search subjects

## Other Features
- Graphical User Interface (GUI)
- Console-based interface
- File storage for saving data
- Fully functional CRUD system

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- File handling
- GUI (Swing)

## Project Purpose
This project was developed as part of coursework to demonstrate Java OOP principles 
and CRUD operations within a small management system.
